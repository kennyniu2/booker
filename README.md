# The Booker
A Go-based REST API + Discord Bot (in progress!) for managing a personal book collection.

# Features
Book Management: Add and manage books in a centralized database

User Collections: Track books users are reading, have completed, or want to read

Ratings & Reviews: Rate books and write reviews

Progress Tracking: Monitor reading progress

# Tech Stack
Language: Go 1.24.1

Framework: Gin (Web framework)

Database: PostgreSQL

ORM: sqlx
